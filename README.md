# Графический редактор

Приложение предназначено для добавления и перемещения графических фигур на рабочем поле.

## Функциональные возможности

- добавление фигур на холст
- выбор цвета фигуры
- перемещение фигур мышью
- отображение фигур на рабочей области

## Архитектура приложения

Приложение построено по шаблону MVC:

- Model - классы фигур и их свойства
- View - графический интерфейс JavaFX
- Controller - обработка действий пользователя

## Реализация

В проекте реализованы:
- фигуры: круг, прямоугольник, треугольник
- выбор цвета через ColorPicker
- отрисовка на Canvas
- перемещение фигур с помощью мыши

## Запуск

Запуск выполняется через класс MainApplication.
